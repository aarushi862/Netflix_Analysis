This project focuses on analyzing Netflix's dataset to extract meaningful insights about movies and TV shows. Using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn, the project explores various aspects such as content distribution, release trends, popular genres, and ratings.

Key objectives:
- Analyze the distribution of Movies vs TV Shows
- Identify the most common genres on Netflix
- Study content trends over the years
- Handle missing data and perform data cleaning
- Visualize insights using graphs and charts

Tools & Technologies:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

This project is a great example of Exploratory Data Analysis (EDA) and helps in understanding real-world datasets.
